# Password Generator 🔐

A simple and responsive password generator built while following
**Chai aur React** playlist by Hitesh Choudhary.

- Not live yet

## 🛠️ Built With

- React.js
- Tailwind CSS
- Vite

## ✨ Features

- Generate random passwords
- Adjust password length with a slider
- Include/exclude numbers and special characters
- Copy to clipboard with one click
